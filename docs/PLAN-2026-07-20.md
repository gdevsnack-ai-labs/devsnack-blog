# DevSnack Redesign — Implementation Plan

> "AI와 함께하는 블로그 관리" — DevSnack Lab, 디자인 개선 및 구조 공개
> 생성: 2026-07-20 | 작업자: Hermes Agent (AI) + 사장님 (검증/피드백)

---

## 🎯 Overview

### 목표
1. **Lab 정체성 강화** — DevSnack을 단순 블로그가 아닌 "AI 연구실"로 브랜딩
2. **LNB 도입** — PC 좌측 사이드 네비게이션으로 탐색성 개선
3. **Lab 페이지** — 실험(Experiment) 중심의 프로젝트 허브
4. **랜딩 리디자인** — Lab 통계 + 프로젝트 프리뷰 통합
5. **소스 공개 대비** — 구조화된 코드, `.env.example`, README 정리

### 철학
- **"발전하는 블로그"** — 미완성도 공개, 진행률 표시
- **"Lab = 콘텐츠 허브"** — 프로젝트 하나 = 블로그 시리즈 + YouTube + GitHub 연결
- **"AI와 함께"** — AI가 설계/코딩, 사람이 검증/방향 결정. 이 과정 자체가 콘텐츠

### 확장안
```typescript
interface Experiment {
  id: string
  name: string
  description: string
  progress: number
  color: ExperimentColor         // 'blue' | 'green' | 'orange' | 'purple'
  status: ExperimentStatus       // '진행중' | '완료' | '예정' | '보류' | '미정'
  isDummy?: boolean
  
  // 신규 필드
  category: 'running' | 'planning' | 'completed'  // Lab 페이지 그룹핑용
  startedAt: string                                // '2026.07'
  whyText: string                                  // "왜 만들었는가" (2~3줄)
  nextGoals: string[]                              // ["Visualization", "Self Play"]
  
  // 콘텐츠 연결
  blogPosts?: string[]                             // ['/devsnack/ai-omok-experiment']
  youtubeVideos?: string[]                         // YouTube video IDs
  githubUrl?: string                               // GitHub 저장소 URL
  
  // 타임라인
  timeline: Array<{
    name: string
    status: '완료' | '진행중' | '예정' | '예약'
    date?: string                                  // '2026.07'
  }>
}
```

## 📋 Phase 2: Lab Page (`/lab`)

### LabDashboard

```tsx
// Supabase에서 실시간 집계
interface LabStats {
  runningProjects: number      // status='running'인 Experiment 수
  totalPosts: number           // posts 테이블 count (status='live')
  totalVideos: number          // youtube-latest.json count
  modelsTested: number         // benchmark DB count
  benchmarkRuns: number        // benchmark DB rows
  lastUpdated: string          // today
}
```

**데이터 소스:**
- `experiments.ts` → 실험 상태/갯수
- `supabase.from('posts').select('id', { count: 'exact', head: true })` → 전체 글 수
- `public/data/youtube-latest.json` → 영상 수
- `public/data/benchmark-summary.json` → 벤치마크 통계 (추가 필요)

### 상태별 그룹

```tsx
const GROUPS = [
```
